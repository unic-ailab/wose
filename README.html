How to run the model

Code repo for the paper titled "Word and Sentence Level Representations for Implicit Aspect Extraction"

1: The code-base is set to run without additional path settings, if it is downloaded and placed at the downloads folder as follows:

ñ       wose-master

|        datasets

|        embeddings

|        output-data

|        source-code

|        README.html

2: The embeddings folder must contain the crawl-300d-2M-subword.zip

3: The train_wose.py in the source-code folder is the main file to load & train the model.

Requirements

  • tensorflow version: '1.15.4'
  • numpy version: '1.16.1'
  • sklearn version: '0.19.0'
  • nltk version: '3.2.4'

4: The configs.py in the source-code folder contains the main parameter settings of the model

Valid input dataset names in configs.py setup

    • Laptop 14 dataset: filename_train=Laptop_Train14.xml, filename_test=Laptop_Test14.xml, filename_preset=LAPTOP14.txt
    • Restaurants 14 dataset: filename_train=Restaurants_Train14.xml, filename_test=Restaurants_Test14.xml, filename_preset=RESTAURANTS14.txt
    • Restaurants 15 dataset: filename_train=Restaurants_Train15.xml, filename_test=Restaurants_Test15.xml, filename_preset=RESTAURANTS15.txt
    • Restaurants 15 dataset: filename_train=Restaurants_Train16.xml, filename_test=Restaurants_Test16.xml, filename_preset=RESTAURANTS16.txt
